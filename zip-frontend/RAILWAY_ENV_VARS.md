# Railway Frontend Environment Variables

Add ALL of these in Railway dashboard → Service → Variables

```
NEXT_PUBLIC_API_URL=https://your-backend-url.railway.app
NEXT_PUBLIC_WALLETCONNECT_PROJECT_ID=your_key
```

**Note:** 
- Replace `your-backend-url.railway.app` with the actual backend service domain from Railway
- Get `NEXT_PUBLIC_WALLETCONNECT_PROJECT_ID` from WalletConnect Cloud
